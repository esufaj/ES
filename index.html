SEE MY WORK

Twitter Streaming Text Analytics

In this project, I design and implement a Twitter streaming app that identifies Twitter trends based on hashtag popularity and calculates a sentiment analysis on Tweets in order to determine the contextual polarity/ emotional reaction of the user. Both are plotted on graphs in real-time. This project uses tools such as the Twitter Developer API, Apache Spark, AJAX, HTML, and JavaScript. Tested on Docker using Hadoop.

Distributed Text Analytics

In this project, I implement numerous MapReduce algorithms to perform distributed analytics on textual data such as N-grams, business frequency/trends, and inverted indexes like the operations performed in the fields of computational linguistics. Tested on Docker using Hadoop.

Data Analytics

In this project, I parse a large dataset from Yelp spanning 4 countries and 11 metropolitan areas in order to compute descriptive analytics, network analytics, frequency distributions, and social networks. This data is then recorded used to make informed data driven decisions.

C-Shell Interface

In this project, I create a C program which serves as a shell interface that accepts user commands and then executes each command in seperate processes. This shell replicates many of the processes native to the command line as well as adding some new ones.

Crypto-Currency

In this project, a group of friends and I are in the process of creating a crypto currency which will become public on the BNB network. We are also in the process of creating a website to advertise the coin. The general framework and burn features were created using Solidity.

Questeme Game Automation

In this project, I use the JDBC to create automation which will send requests to a PSQL database in order to update quests, loot, themes, and timeframe entries that have been collected from players of the game. Previously, these updates were being inserted manually, but as the player base grows this is not a sustainable approach and thus we automate.
